Assignment 3

This assignment was done in collaboration with Casey Currey-Wilson.

The goal of this assignment is to write a program that will convert input from a Bézier surface representation (a .bez file with patches made with 4 curves with 4 control points) to polygon representation (using either uniform or adaptive technique, both of which are implemented) and display it with OpenGL.

Our program is done in Python 2.7, so to run it, simply use the command:
python bezier.py input.bez 0.1 -a

Replacing
input.bez
with your desired input file name, and
0.1
with your desired subdivision parameter. The
-a
flag is optional and indicates adaptive subdivision. Uniform tessellation is the default.

There are various commands that can be used when the objected is rendered in a window to view the object differently (at various angles, different shading etc.). They are detailed below:
  • "s" - Toggles between flat and smooth shading
  • "w" - Toggles between filled and wireframe mode
  • "←, ↑, →, ↓" - Rotates object
  • "⇧" (shift) + "←, ↑, →, ↓" - Translates object
  • "+, -" - Zooms in and out; "=" also works for zooming in


The files used in the examples below are test.bez, teapot.bez, and simple.bez. The first two were given as example files with the assignment. The third one we made by writing simple coordinates.
